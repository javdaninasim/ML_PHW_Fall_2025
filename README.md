## Assignments

### `ML_PHW1` — Foundations & Linear Models
- Linear regression: closed-form solution and gradient descent
- Logistic regression for binary classification
- Regularization: L1 (Lasso) and L2 (Ridge)
- Bias-variance trade-off and learning curves

### `ML_PHW2` — Tree-Based & Ensemble Methods
- Decision trees: ID3 / CART with pruning
- Random forests and bagging
- Gradient Boosting (XGBoost / AdaBoost)
- Feature importance and out-of-bag evaluation

### `ML_PHW3` — Support Vector Machines & Kernel Methods
- Hard-margin and soft-margin SVM
- Kernel trick: RBF, polynomial, linear kernels
- Hyperparameter tuning with cross-validation
- Comparison with logistic regression

### `ML_PHW4` — Unsupervised Learning & Dimensionality Reduction
- K-means clustering and convergence analysis
- Principal Component Analysis (PCA)
- t-SNE for high-dimensional data visualization
- Clustering evaluation: silhouette score, elbow method

### `ML_PHW5` — Neural Networks & Deep Learning
- MLP implementation with backpropagation (from scratch)
- Convolutional Neural Networks (CNNs) with PyTorch
- Batch normalization, dropout, early stopping
- Training on image classification benchmarks

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/javdaninasim/ML_PHW_Fall_2025.git
   cd ML_PHW_Fall_2025
   ```
2. Install dependencies:
   ```bash
   pip install numpy pandas scikit-learn torch matplotlib seaborn jupyter
   ```
3. Navigate to any homework folder and launch Jupyter:
   ```bash
   jupyter notebook
   ```
